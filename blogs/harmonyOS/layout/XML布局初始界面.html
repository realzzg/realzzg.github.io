编写第一个鸿蒙XML布局

zzg 2021-11-19 鸿蒙-布局XML

# 创建项目

在安装好DevEco-Studio后选择，选择Empty Ability(Java)，点击Next

在Project Type下选择Application，在Device Type下选择Phone，或者其他你想要创建的项目(可多选)。

# 布局文件

打开entry/src/main/resource/base/layout/ability_main.xml 文件会含有以下代码：

<?xml version="1.0" encoding="utf-8"?>
<DirectionalLayout
    xmlns:ohos="http://schemas.huawei.com/res/ohos"
    ohos:height="match_parent"
    ohos:width="match_parent"
    ohos:alignment="center"
    ohos:orientation="vertical">

    <Text
        ohos:id="$+id:text_helloworld"
        ohos:height="match_content"
        ohos:width="match_content"
        ohos:background_element="$graphic:background_ability_main"
        ohos:layout_alignment="horizontal_center"
        ohos:text="$string:mainability_HelloWorld"
        ohos:text_size="40vp"
        />

</DirectionalLayout>

# 代码说明

<DirectionalLayout/>
1

将一组组件按照水平或者垂直方向排布，能够方便的对齐布局内的组件，它有以下属性：

  • alignment: 对齐方式，有取值：left top horizontal_center start等，其中最常用的是horizontal_center vertical_center center，水平居中、垂直居中、居中。
  • orientation:子布局排列方式，有取值：horizontal vertical，水平方向和垂直方向两种布局
  • width height，布局大小，在这里统一设置为_match_parent_
  • background_element：背景元素，可以设置颜色(#000000)等
<Text/>
1

文本组件

id：自定义当前组件的ID；

"$graphic:background_ability_main"：资源的相对路径

text_size="40vp"

vp：虚拟像素单位，针对一台设备针对应用而言所具有的虚拟尺寸，区别于像素单位，vp可以在不同的设备上提供相同的视觉体验

fp：字体像素单位，默认情况下与_vp_相同，在设置更大字体后，1 fp = 1vp * scale

px：svg标准，采用px作为单位

# 添加button按钮

将DirectionalLayout改为DependentLayout，DependentLayout有更加灵活的排布方式方式，每个组件可以指定相对于其他同级元素的位置，或者制定相对于父组件的位置。并且删除两行代码：

<DirectionalLayout
		ohos:alignment="center"
		ohos:orientation="vertical"/>
1
2
3

添加：

 <DependentLayout
		ohos:alignment="horizontal_center"/>
1
2

详细代码如下：

<?xml version="1.0" encoding="utf-8"?>
<DependentLayout
    xmlns:ohos="http://schemas.huawei.com/res/ohos"
    ohos:height="match_parent"
    ohos:width="match_parent"
    ohos:alignment="horizontal_center" >

    <Text
        ohos:id="$+id:text_helloworld"
        ohos:height="match_content"
        ohos:width="match_content"
        ohos:background_element="$graphic:background_ability_main"
        ohos:center_in_parent="true"
        ohos:text="$string:mainability_HelloWorld"
        ohos:text_size="40vp"
        />

    <Button
        ohos:id="$+id:button_next"
        ohos:height="match_content"
        ohos:width="match_content"
        ohos:background_element="$graphic:button_element"
        ohos:top_padding="8vp"
        ohos:bottom_padding="8vp"
        ohos:center_in_parent="true"
        ohos:left_padding="70vp"
        ohos:margin="10vp"
        ohos:right_padding="70vp"
        ohos:text="Next"
        ohos:text_color="white"
        ohos:text_size="18vp"
        ohos:align_parent_bottom="true"
        />

</DependentLayout>

# 代码说明

padding：设置元素边框和元素内容之间的空间，即上下左右的内边距

margin:设置元素与元素之间的间距

align_parent_bottom：对齐方式在相对于父组件的底部，界面的布局是多种设置相互叠加的效果，不同的组件设置可能相互冲突

# 添加按钮背景

创建entry/src/main/resource/base/graphic/button_element.xml文件

添加以下代码：

<?xml version="1.0" encoding="utf-8"?>
<shape
    xmlns:ohos="http://schemas.huawei.com/res/ohos"
    ohos:shape="rectangle">
    <solid
        ohos:color="#000DFF"/>
    <corners
        ohos:radius="100vp"/>
</shape>
